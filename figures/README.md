This folder contains the figures, Python scripts, datasets, and additional metadata (README and description files) used to generate the figures related to the NMR-ML model in the manuscript and supporting information documents of ‚Äú***Neural Networks-Enabled Insights Into Quantum Effects on Structure and <sup>13</sup>C NMR of C<sub>60</sub> Fullerene***‚Äù.

## Requirements

To reproduce the figures and analyze the data, you will need the following:
- **Python 3.x**
- **Matplotlib**
- **NumPy**
- **Pandas**
- **SciPy**
- **Sklearn**
- **os**
